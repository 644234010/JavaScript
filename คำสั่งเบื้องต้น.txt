var = ไว้กำหนดตัวแปล
let = กำหนดตัวแปล ใช้ได้ในขอบเขตเท่านั้น

alert = ใช้แสดงป๊อบอัพด้านบนให้เป็นค่าต่างๆ 

console.log = ใช้แสดงค่า เหมือนกับprintf 

prompt  = ใช้แสดงป๊อบอัพให้ป้อนข้อมูลได้

Math.floor() = ใช้เพื่อปัดลง (round down) ค่าทศนิยมของจำนวนที่กำหนด. นั่นคือ, มันจะลดค่าทศนิยมลงให้เหลือเพียงจำนวนเต็มที่ใกล้ที่สุดที่น้อยกว่าหรือเท่ากับค่าที่กำหนด.
Math.random สุ่มตัวเลข0,1

innerhtml = ใส่โค๊ดได้
innerText = ใส่ได้แค่ข้อความ

if-else = เงื่อนไข

loop for while   = วนซ้ำ

parseInt = แปลงข้อมูลจำนวนเต็ม
toFixed() = จำกัดแสดงทศนิยม
RETURN  = สงค่ากลับ
